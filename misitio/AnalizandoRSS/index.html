Estructura y elementos de RSS 2.0:

  1. Estructura principal de un documento RSS

    2. 
                <?xml version="1.0" encoding="ISO-8859-1" ?>
                <rss version="2.0">
                <channel>
                <title></title>
                <link></link>
                <description></description>
                    <item>
                    ... <!-- Tantos elementos como entradas queramos -->
                    </item>
                </channel>
                </rss>

  2. Indica cuántos elementos puede haber en un documento RSS

    3. Solo puede haber 1

  3. Indica los elementos requeridos en <channel> (son tres)

    4. 
                <title></title>
    
                <ink></link>
    
                <description></description>

  4. Indica los elementos secundarios en <channel> (son 16)

    5. language, language, managingEditor, webMaster, pubDate, lastBuildDate, category, generator, docs,
    cloud, ttl, image, textInput, skipHours, skipDays

  5. Elementos item

    6. title, link, description, author, category, comments, enclosure, guid, pubDate, source

  6. Indica el elemento que añade un elemento multimedia. Indica sus tres atributos

    7. 
                <enclosure ulr=""></enclosure>
    
                <enclosure lenght=""></enclosure>
    
                <enclosure type=""></enclosure>

Estructura y elementos de Atom 1.0:

  1. Estructura principal de un documento Atom

    2. 
                <?xml version="1.0" encoding="utf-8"?>
                <feed xmlns="http://www.w3.org/2005/Atom">
                <title>Example Feed</title>
                <link href="http://example.org/"/>
                <updated>2003-12-13T18:30:02Z</updated>
                <author>
                    <name>John Doe</name>
                </author>
                <id>urn:uuid:60a76c80-d399-11d9-b93C-0003939e0af6</id>
                <entry>
                    <title>Atom-Powered Robots Run Amok</title>
                    <link href="http://example.org/2003/12/13/atom03"/>
                    <id>urn:uuid:1225c695-cfb8-4ebb-aaaa-80da344efa6a</id>
                    <updated>2003-12-13T18:30:02Z</updated>
                    <summary>Some text.</summary>
                </entry>
                </feed>

  2. Indica cuántos elementos <feed> puede haber en un documento Atom

    3. Solo 1.

  3. Indica los elementos requeridos en <feed>

    4. 
                <title>Example Feed</title>
                <updated>2003-12-13T18:30:02Z</updated>
                <id>urn:uuid:60a76c80-d399-11d9-b93C-0003939e0af6</id>

  4. Indica los elementos recomendados en <feed>

    5. 
                <author></author>
    
                <link></link>

  5. Indica los elementos opcionales en <feed>

    6. 
                <category></category>
    
                <contributor></contributor>
    
                <generator></generator>
    
                <icon></icon>
    
                <logo></logo>
    
                <rights></rights>
    
                <subtitle></subtitle>

  6. Indica los elementos requeridos en <entry>

    7. 
                <id></id>
    
                <title></title>
    
                <updated></updated>

  7. Indica los elementos recomendados en <entry>

    8. 
                <author></author>
    
                <content></content>
    
                <link></link>
    
                <summary></summary>

  8. Indica los elementos opcionales en <entry>

    9. 
            <category></category>
    
            <contributor></contributor>
    
            <published></published>
    
            <rights></rights>
    
            <source></source>
